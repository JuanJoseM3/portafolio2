Sobre mi

Actualmente me encuentro terminando mi curso de aprendizaje en Academlo para ser Programador Web Full Stack. Entre los conocimientos adquiridos hasta el momento, se encuentran HTML, CSS, JavaScript, NodeJS, express, así como la realización de diversos proyectos con estas tecnologías y el consumo de APIs con React.

Consciente del constante cambio del mercado y las tecnologías actuales, soy un entusiasta del aprendizaje y de la búsqueda de nuevas técnicas para la resolución de las necesidades de las aplicaciones web.

Considero que el aspecto más importante en el desarrollo de aplicaciones web es entender previamente todos los requerimientos del cliente para poder trazar un plan de trabajo que lleve a la solución más óptima.

Habilidades y Servicios

Front-End

Manejo de HTML y CSS para la estructuración y visualización de páginas web, según las necesidades y preferencias del cliente. Diseño de la aplicación bajo la metodología de Mobile First y Responsive Web Design

Back-End

Implementación de Javascript y NodeJS para la declaración de funciones y desarrollo de lógica para páginas web. Manejo del sistema PostgreSQL para la gestión de bases de datos y desarrollo.

Proyectos

API Rick and Morty

Consumo de la API de Rick and Morty mediante React JS para retornar al usuario una interfaz en la que puede consultar los detalles de todos los personajes de la serie. Incluye funcionalidad de barra de búsqueda para filtrar personajes de acuerdo a su universo.

API Pokemon

Implementación de React para la renderización de la API de Pokemon. Se listan los pokemon haciendo uso de la paginación según la cantidad de pokemon que el usuario desee ver por página, a la vez que se tiene la opción de buscar los pokemon según su tipo, nombre o ID.

Lógica Back-End para e-commerce

Definición de la lógica, funciones y relaciones de modelos en la base de datos para guardar la información de usuarios del e-commerce, sus productos a la venta y el historial de compras de usuarios. Proyecto desarrollado a través de NodeJS y probado en la plataforma Postman.
